Gamio v1.0 Documentation

1. Upload and Extract files

Upload archive which you download from our website to your hosting via File Manager or some FTP client.
Then extract it, copy or move files from PHP Script directory to root directory of your hosting.
If you use sub-directory move files there, but need to change path in .htaccess file. Open .htaccess file and where is line "RewriteBase /" change it to "RewriteBase /your_directory/".

With this we finish this step of installation, click here to continue with next.
